# Raid Simulation Results: Longbow DD (PIERCE) — Common vs Legendary

**Изменения от предыдущего теста:**
- DD оружие: Crossbow (BLUNT, range 4, 370 atk, 5 crit) → **Longbow (PIERCE, range 3, 380 atk, 3 crit)**
- DD модификаторы: убран SELF_HEAL на штанах → заменён на **COUNTER_ATTACK** (стакается с body до rank V)
- Оставшиеся модификаторы: DOUBLE_ATTACK (longbow), COUNTER_ATTACK×2 (body+pants), FEINT, RETREAT, THORNS

**Персонажи:**
- **Танк** (FRONT): Mace + Tower Shield + Cuirass + Greaves + Sabatons + Great Helm + Gauntlets — PLATE
- **ДД** (BACK): Longbow + Wizard Robe + Arcane Chausses + Arcane Boots + Circlet + Arcane Gloves — ARCANE

**Параметры:** 1000 итераций, уровни 5/10/15, все 4 типа рейдов

## Сравнение с предыдущим тестом (Crossbow BLUNT → Longbow PIERCE)

### Win Rate Delta (Longbow - Crossbow), Common

| Рейд | Lvl 5 | Lvl 10 | Lvl 15 |
|------|-------|--------|--------|
| WolfPack | 94.4 → 95.7 (**-1.3**) | 65.8 → 73.2 (**-7.4**) | 24.2 → 30.9 (**-6.7**) |
| ZombieHorde | 99.2 → 99.8 (**-0.6**) | 76.5 → 92.9 (**-16.4**) | 7.6 → 42.4 (**-34.8**) |
| MyconidColony | 100.0 → 100.0 (0) | 99.8 → 98.6 (**+1.2**) | 88.5 → 86.0 (**+2.5**) |
| MaggeeseFlock | 99.9 → 99.9 (0) | 97.9 → 97.4 (**+0.5**) | 84.9 → 65.1 (**+19.8**) |

**Длинный лук (PIERCE) катастрофически слаб против ZombieHorde** (-34.8% на уровне 15!), потому что PIERCE имеет ×1.1 эффективную защиту против PLATE брони зомби, тогда как BLUNT имел ×0.75.

Зато длинный лук **значительно лучше** против MaggeeseFlock (+19.8% на уровне 15) — PIERCE режет CLOTH броню гусей с ×0.75 эффективной защиты.
